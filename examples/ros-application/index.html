Teleop Demo

This custom view demonstrates how arbitrary bi-directional communication over real-time custom data channels works.

In this custom web view:

  • Sending data to the robot application from a custom web application
  • Listening to data from the robot application in a custom web applicaton
In the associated python robot application:
  • Sending data to a custom web application from the robot application
  • Listening to data from a custom web application in the robot application

See the associated Python adapter for robot application examples. https://github.com/FormantIO/toolkit/examples/custom-data-channel-communication/adapter.py

Check out https://github.com/FormantIO/toolkit for more general information!
